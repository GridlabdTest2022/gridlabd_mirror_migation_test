Welcome GridLAB-D User

This is probably not what you are looking for. Your query is likely missing information about the code object you wish to download.

This endpoint is used to download code objects in the GridLAB-D repository. For example, the command

% curl https://code.gridlabd.us/master/README.md
will return the master branch's version of the README.md file.

Note that unauthenticated access is rate limited. Please see GitHub API Rate Limits for more information.
